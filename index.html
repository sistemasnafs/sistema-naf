💾 Backup completo do sistema
Use o arquivo de backup JSON para guardar tudo: NAFs, responsáveis e configurações. Esse formato preserva 100% dos dados e é o recomendado para atualizações do sistema. O CSV é útil para abrir no Excel ou compartilhar com outras pessoas.

📤 Restaurar dados
Restaure dados de um backup anterior. Atenção: ao restaurar, você pode escolher entre substituir tudo ou adicionar as NAFs ao que já existe no sistema.
📦 Restaurar do arquivo JSON (backup completo)
Restaura NAFs + responsáveis + configurações.
Formato: backup_naf_AAAA-MM-DD.json
📊 Importar do arquivo CSV (só NAFs)
Para reimportar NAFs exportadas pelo sistema. As colunas devem seguir o mesmo padrão do CSV exportado.
Formato: controle_nafs_AAAA-MM-DD.csv
💡 Recomendação: Faça um backup JSON pelo menos uma vez por semana e antes de qualquer atualização do sistema. Guarde os arquivos no Google Drive para segurança extra.
⚡ Integração com Supabase
🔗 Credenciais do projeto

📊 Status da conexão
Conecte para ver o status detalhado das tabelas.

🤖 Configuração de IA (extração automática de NAFs)
Configure a API de inteligência artificial para extrair dados automaticamente de fotos ou PDFs de NAFs. Aceita qualquer provedor compatível com o formato OpenAI (ChatGPT, Claude, Groq, OpenRouter, Ollama, etc.).
Gemini: https://generativelanguage.googleapis.com · OpenAI: https://api.openai.com/v1/chat/completions
